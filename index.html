Программа визита

например: «демонтаж», «425-од», «бот»
10.09.2025 (среда)
Кабинет • Регламенты и обмен опытом

14:00–14:30 • Формат работы, представление участников

14:30–15:20 • Демонстрация регламентов и приказов

Обзор ключевых документов: состав комиссий, алгоритмы взаимодействия, регламент доступа к ВЛ.
  • Приказ 08.04.2025 №245-од — состав комиссии и график проверки филиалов до сентября 2026 г.
  • Приказ 17.06.2025 №425-од — алгоритм взаимодействия при инвентаризации ВОЛС; роли, телеграм-бот, ответственность.
  • Приказ 27.06.2024 №476-од — регламент оказания услуг по предоставлению доступа к ВЛ для размещения ВОЛС.

15:20–17:00 • Обмен опытом

  • Положительная практика по гарантийным письмам (утверждённый формат).
  • Претензионная работа при неподписанных договорах.
  • Памятка персоналу по демонтажу ВОЛС.
  • Работа с контрагентами и сопровождение: просроченная задолженность, организация судебной работы, регламент выявления бездоговорного ВОЛС (взаимодействие блоков), актуализация реестров и структуры сети (изменения по 425-од), порядок выявления бездоговорного подвеса (передача практик).
  • Демонстрация телеграм-бота (сценарии, роли, выгрузки, отчётность).
11.09.2025 (четверг)
Выезды • Контроль размещения • Действия при БД ВОЛС

08:00–13:00 • Контроль размещения ВОЛС на ВЛ

  • Проверка договорного ВОЛС: сверка с реестрами договоров.
  • Демонстрация ведения учёта договорного ВОЛС (мобильные формы/бот).

13:00–16:30 • Действия при обнаружении бездоговорного ВОЛС

  • Фиксация, уведомления, оформление документации.
  • Взаимодействие подразделений по 425-од.
  • Принятие решения: уведомления / демонтаж / оформление договора размещения.

16:30–17:00 • Подведение итогов дня

12.09.2025 (пятница)
Итоги • Протокол • Выезд группы

07:00–07:30 • Подведение итогов и обратная связь

Обсуждение тиражирования практик, формирование пунктов протокола.

07:30–08:10 • Выезд представителей рабочей группы ПАО «Россети»
